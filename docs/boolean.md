# Boolean

 A boolean is either `true` or `false`.

## Operators

| Expression | Description
| --- | ---
| `b1 == b2` | Equality
| `b1 != b2` | Inequality
| `b1 && b2` | Logical AND
| `b1 \|\| b2` | Logical OR
| `!b` | Logical NOT

## `b.toNumber()`

 Returns `1` if `b` is `true`. Returns `0` if `b` is `false`.

## `b.toString()`

 Returns a string representing `b` (i.e. `"true"` or `"false"`).

## `b.type()`

 Returns `"boolean"`.
